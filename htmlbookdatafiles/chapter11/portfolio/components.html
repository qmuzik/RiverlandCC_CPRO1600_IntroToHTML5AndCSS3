Components

Central Processing Unit (CPU)

The CPU is also known as the brain of the Computer. It's job is to calculate all the equations and instructions that occur when a program is running. Each CPU has Cores and Threads. The more cores the more work can be done without stressing the pc. Threads allow the cores to transfer data faster between each core.

Random Access Memory (RAM)

The RAM is used for storing data that is being created and used while various computer programs run. The data is deleted after the PC turns off. When choosing RAM, there are three attributes to know about; Speed in Mhz, Size in GB and Channels. Remember, the more RAM you have, the more programs can run on your computer at once.

Storage Drives

Depending on what you want to be stored on your computer, you will need storage. There are two options when choosing a drive; Hard Drive or Solid State Drive. A hard drive is a spinning disc that write data while being used and have large capacities. Solid States use circuit boards to store memory via electrical current, which results in faster loading speeds.

Graphics Processing Unit (GPU)

Are you a gamer or do you enjoy video editing? If you said Yes to both of those, You will need a good GPU. The GPU is used for processing computer graphics faster. When choosing a GPU, remember to look: Brand (NVIDIA or AMD), Video Memory Size (VRAM) and Clock Speed (MHz). The other options besides a dedicated GPU are an onboard GPU such as Intel Graphics.
